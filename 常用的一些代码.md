### 设置返回数据为变量

```
pm.test("Status code is 200", function () {
    pm.response.to.have.status(200);
});

// 把responseBody转为json字符串
var res = JSON.parse(responseBody);
// 设置环境变量token，供后面的接口引用
pm.environment.set("token", res.data);
```
